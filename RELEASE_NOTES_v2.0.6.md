# TeacherPro v2.0.6

## Bug Fixes

### Material Link Context Menu (Fixed)
- **Fixed material link right-click menu** — Inserted materials in lesson plans now properly display the context menu with Preview, Open in Default App, Reveal in Finder, and Remove Link options.
- **Technical fix**: The context menu is now rendered using React Portal to escape the overflow-hidden container that was clipping it in previous versions.
