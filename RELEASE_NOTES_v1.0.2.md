# Release Notes - TeacherPro v1.0.2

### Version 1.0.2 (April 14, 2026)

**Hotfix:**
- **Mac Startup Window Behavior:** Changed the startup window state from "Fullscreen" to "Maximized". This avoids a clunky MacOS behavior where switching to a native fullscreen app swaps to a separate virtual desktop workspace. Your app will now neatly fill your screen natively within your main workspace.
