# TeacherPro v2.0.4 Release Notes

**Release Date:** April 19, 2026

## What's New

### Method Bank Expansion
20 new teaching methods have been added to the Method Bank, bringing the total to **103 methods**. These methods cover assessment strategies, scaffolding techniques, arts integration, and STEM-specific approaches:

- **All-Student Response** — Every student responds simultaneously using mini-whiteboards, hand signals, or gestures
- **Peer Teaching Protocol** — Students teach concepts to each other in structured pairs
- **Self-Regulation Check-in** — Students assess their own understanding and set improvement goals
- **Purposeful Questioning Sequence** — Planned questions that reveal and advance student thinking
- **Productive Struggle Support** — Supporting students through challenging tasks without rescue
- **Sentence Stems for Academic Talk** — Language frames for structured academic discussion
- **Frontloading Vocabulary** — Pre-teaching essential terms with visuals
- **Graphic Organizer with Word Bank** — Visual organizers paired with key vocabulary
- **Comprehensible Input Routine** — Adjusting speech, visuals, and gestures for accessibility
- **Tableau (Frozen Scene)** — Students create frozen pictures depicting concepts
- **Reader's Theatre** — Scripted oral reading performance without memorization
- **Shadow Puppet Narrative** — Using shadow puppets to explain academic content
- **Error Analysis Protocol** — Students identify and correct errors in worked examples
- **Number Talk** — Mental math discussion comparing solution strategies
- **Claim-Evidence-Reasoning** — Scientific argumentation structure for labs
- **Notice & Wonder** — Open-ended exploration before formal instruction
- **Which One Doesn't Belong?** — Open-ended reasoning with multiple valid answers
- **Co-Creating Success Criteria** — Students collaborate to define quality standards
- **Two Stars and a Wish** — Structured peer feedback protocol
- **Traffic Light Self-Assessment** — Students signal understanding with colored cards

### Trash Auto-Clear
New automatic trash cleanup feature in Settings > Advanced:
- Set trash to auto-clear after **7, 14, 30, or 90 days**
- Default: **30 days**
- Option to disable ("Never") for manual-only deletion
- Cleanup runs automatically when vault refreshes
