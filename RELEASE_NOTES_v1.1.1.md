# Release Notes - TeacherPro v1.1.1

### Version 1.1.1 (April 15, 2026)

This release focuses on UI/UX quality improvements for daily use: cleaner settings organization, stronger theme readability, and better visual consistency across sidebar controls.

## Highlights

- Settings got a structural refresh:
  - New section tabs: Appearance, Defaults, Advanced.
  - Stable panel height so tab switching no longer causes jumping/resizing.
- Theme and writing surfaces are now more flexible:
  - Light mode readability overhaul across menus, panels, and previews.
  - Independent paper tone for Lesson Plans and Mindmaps (White/Dark), separate from global app theme.
- Sidebar control polish:
  - Consistent spacing between collapsible categories.
  - Unified size/style for section action buttons (search/add/upload controls).
- Editor formatting reliability:
  - Ordered list numbering now renders correctly.

## Versioning

- App version updated to `1.1.1` across:
  - `package.json`
  - `src-tauri/Cargo.toml`
  - `src-tauri/tauri.conf.json`
- Build workflow release tag format aligned to `v__VERSION__` for cleaner `v1.1.1` release tagging.
