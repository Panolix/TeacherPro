# TeacherPro v2.0.2

Patch release with font sizing improvements.

## Changed
- Updated default font sizing in the lesson plan editor from 12pt to 11pt for better readability
- Table headers now inherit the default font size instead of using a smaller hardcoded value
- Adjusted heading sizes proportionally (H1: 16pt, H2: 13pt, H3: 12pt) to maintain hierarchy at the smaller base size
